# Gestión Educativa React

Sistema de gestión académica full-stack con autenticación por roles, seguimiento de asistencia y dashboard de alumnos en riesgo.

## Stack

- **Backend:** Node.js + Express + SQLite (better-sqlite3)
- **Frontend:** React + Vite + Tailwind CSS + shadcn/ui
- **Auth:** JWT con roles `admin` y `docente`
